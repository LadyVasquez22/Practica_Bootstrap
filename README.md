# PracticaBootstrap

## Descripción general

Este proyecto corresponde a la práctica de integración de Bootstrap 5 en un portafolio web.  
Se creó una página de proyección estudiantil y profesional, vinculada con las páginas anteriores del portafolio.

La página presenta metas académicas, habilidades por desarrollar, áreas tecnológicas de interés, proyectos futuros y un plan de mejora personal y académico.

## Tecnologías utilizadas

- HTML5
- CSS3
- Bootstrap 5.0.2
- Font Awesome
- GitHub

## Instrucciones para visualizar el sitio

1. Descargar o clonar el repositorio.
2. Abrir la carpeta `PracticaBootstrap`.
3. Abrir el archivo `index.html` en el navegador.
4. Desde el menú de navegación, ingresar a la página de **Proyección**.

También se puede abrir directamente el archivo:

pages/cto/proyeccion.html
